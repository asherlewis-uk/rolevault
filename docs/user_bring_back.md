### Deviations
- **Database port**: The local RoleVault Postgres container is exposed on port `5433` (not 5432). All backend commands and `.env` were adjusted to use `localhost:5433`.
- **Alembic stamp required**: The database was already initialized outside of Alembic, so `alembic stamp acf1d10fbf13` was run before generating the new migration.
- **requests dependency added**: The new Apple auth endpoint uses `requests` to fetch Apple JWKS. It was not in the original `requirements.txt` and has been added alongside `pyjwt`.
- **Clean migration**: The autogenerated migration included noisy FK reconstructions. It was manually cleaned to contain only the `apple_user_id` column addition and unique constraint.
